Before executing make sure the <ip address> , <wifi id> , <wifi password> fields are replaced with appropriate data.
The server recieves the data so it must be run in the ESP32 microcontroller, store this Server.py file as boot.py in the microcontroller to make the microcontroller automatically execute this code upon turning on.
The client code must be run in a device connected in the same network
Connecting any device or relays to the appropriate pins will make the device remotely accessible
Easly scalable to multiple devices
